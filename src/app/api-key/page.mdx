# API Key

In order to interact with TofuPilot's API, you will need to use a personnal API key. This key will authenticate your API requests and act on your behalf. It is unique per user. {{ className: 'lead' }}

## Accessing your API key {{ id: 'accessing-key' }}

You can access and manage your API key via your profile page, accessible under the settings menu. {{ className: 'lead' }}

<Image src="/profile-page.png" alt="Your profile page" />

## Creating your API key {{ id: 'creating-key' }}

If you haven't created your API key, press the 'Create' button. {{ className: 'lead' }}

<Image src="/api-key-create.png" alt="Create API key" />

## Managing your API key {{ id: 'managing-key' }}

Once created, you can copy your API key and use it to authenticate against the API. {{ className: 'lead' }}

<Image src="/api-key-manage.png" alt="Manage API key" />

<Note>
  If you think your API Key has been compromised, you can regenerate it at any
  time using the 'Regenerate' button.
</Note>

## Set as environment variable {{ id: 'environment-variable' }}

Set your TofuPilot API Key as an environment variable. You only need to do this once.

```bash
# set TofuPilot API Key (using zsh for example)
$ echo 'export TOFUPILOT_API_KEY=[your_key_here]' >> ~/.zshenv

# reload the environment (or just quit and open a new terminal)
$ source ~/.zshenv
```
